168 /**
169  * @brief main_init - open the Display and init the
170  * main->childlist. Also it set the bool run to true.
171  * This one will be used to terminate the main event loop.
172  * main_init() should be called directly after declaration of Xputty.
173  * Any Widget_t which would be created afterwards will be added to the list.
174  * This list is used to check if a Widget_t is valid.
175  * When a Widget_t call quit_widget() it will be removed from the list.
176  * On main_quit() any remaining Widget_t from this list will be destroyed,
177  * to ensure that we leave the memory clean.
178  * This list is also used to check which Window receive a XEvent.
179  * @param *main - pointer to the main Xputty struct
180  * @return void
